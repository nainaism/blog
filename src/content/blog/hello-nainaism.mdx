日々の開発で学んだことや、気になった技術トピックについて共有していく予定です。

## 技術スタックについて

このブログは以下の技術で構築されています:

- [Astro](https://astro.build/) - 高速な静的サイトジェネレーター
- [astro-theme-pure](https://github.com/cworld1/astro-theme-pure) - ミニマルなブログテーマ
- [Cloudflare Pages](https://pages.cloudflare.com/) - ホスティング

### なぜ Astro を選んだか

Astro はコンテンツファーストのフレームワークで、ブログのようなコンテンツ中心のサイトに最適です。ビルド時に不要な JavaScript を除去するため、表示速度が高速です。

## コードブロックの例

TypeScript で簡単な関数を書いてみます:

```typescript
function greet(name: string): string {
  return `こんにちは、${name}さん！`;
}

console.log(greet('Nainai'));
// => こんにちは、Nainaiさん！
```
